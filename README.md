# Simulador de Financiamento

Aplicativo Flutter para simulação de financiamento de bens, permitindo calcular o valor das parcelas e o montante total a ser pago.

---

## Contexto

As taxas de juros elevadas dificultam a aquisição de bens e serviços. Antes de realizar um financiamento (carro, imóvel, eletrodoméstico, etc.), é importante simular os custos envolvidos.

---

## Objetivo

aplicativo que receba:

* Valor do bem
* Número de parcelas
* Taxa de juros mensal
* Taxas adicionais

E exiba:

* Valor da parcela
* Montante total do financiamento

---

## Tecnologias

* Flutter
* VS Code
* Android Studio

---

## Recursos Utilizados

* Interface com Widgets Flutter
* Campos de entrada (`TextField`)
* Botões (`ElevatedButton`)
* Cálculo financeiro
* Atualização de estado (`setState`)

---

## Como executar

1. Clone o repositório
2. Abra no VS Code
3. Abra o terminal (Ctrl + ")
4. Execute:

flutter pub get
flutter run

5. Escolha um emulador ou navegador
